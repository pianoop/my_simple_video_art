# my_simple_video_art

## 사용 언어
Python  
version: 3.9.12  

## 설명  
CV2 라이브러리를 사용하여 예시로 주어진 영상을 외곽선을 검은 색으로, 그 외는 흰 바탕으로 변형하여 영상을 출력하는 코드입니다.  
가장자리를 추출하는 방식으로는 Laplacian함수를 사용하였습니다.
